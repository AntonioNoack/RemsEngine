# MatCap

MatCap (Material Captures) are a cheap rendering method for mobile devices or for previews.  
They don't support any environment lights.

---
https://github.com/nidorx/matcaps/tree/master is an exellent source for MatCap files except that re-colors are missing. You have to do that yourself in Gimp or Photoshop.
